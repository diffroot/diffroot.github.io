在Mac OS X中使用mtr诊断路由节点问题

发表于 2015-06-24   |   分类于 技术   |  

这个工具是从阿里云客服那知道的，当时遇到阿里云CDN的一个节点出现丢包问题，用这个工具诊断路由节点问题。

1.下载地址：http://rudix.org/packages/mtr.html（在园子里下载）

2.下载后运行mtr-0.85-0.pkg进行安装

3.cd /usr/local/sbin ，就会看mtr文件。

4.运行mtr出现提示

-bash: mtr: command not found

解决方法：
alias mtr=/usr/local/sbin/mtr

5.继续运行mtr出现提示
mtr: unable to get raw sockets.

解决方法： 

sudo chown root mtr
sudo chmod u+s mtr

6.运行mtr进行诊断 

mtr common.cnblogs.com

博客迁移成功

发表于 2015-06-23   |   分类于 生活   |  

迁移过程和原因

Linode ，在使用了一年之后，由于续费的不及时，被无情的关闭，连数据资料都没有来得及备份，心里非常的郁闷，不想再写任何东西了。

但是，最近在看各种书籍时，发现还是有必要开辟一个地方来记录下当时翻阅资料的内容和心情。

于是，开始在网上找各种托管的地方，免费空间现在是真不敢尝试了，那些收费的服务质量不好也是不考虑。后来找来找去，想想开始使用gitgub托管吧，虽然不是特别安全，但是胜在本地有同步的备份，以后恢复起来也是比较的方便。

确定了大的方向后，然后就是寻找各种平台架构，在用2天时间尝试了octopress之后放弃，然后花了半天时间安装了hexo框架，非常成熟的技术。

下一步就是开始慢慢再积累内容咯！

感谢Hexo的作者提供这么好、这么方便的技术！
